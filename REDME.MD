## ğŸš€ Installation

### Prerequisites
- Node.js (v16 or higher)
- MongoDB (local or Atlas)
- npm or yarn
- Git

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/cricket-info-website.git
cd cricket-info-website
```

### Backend Setup

```bash
# Navigate to backend folder
cd backend

# Install dependencies
npm install

# Create .env file
touch .env

# Add environment variables (see below)

# Start backend server
npm run dev
```

Backend will run on `http://localhost:5000`

### Frontend Setup

```bash
# Navigate to frontend folder (from root)
cd frontend

# Install dependencies
npm install

# Create .env file
touch .env

# Add environment variables (see below)

# Start development server
npm run dev
```

Frontend will run on `http://localhost:3000`

## ğŸ’» Usage

### Starting the Application

1. **Start MongoDB** (if using local):
```bash
# Windows
net start MongoDB

# Mac/Linux
sudo systemctl start mongod
```

2. **Start Backend**:
```bash
cd backend
npm run dev
```

3. **Start Frontend**:
```bash
cd frontend
npm run dev
```

4. **Access Application**:
   - Frontend: http://localhost:3000
   - Backend API: http://localhost:5000

### Creating an Admin User

1. Register a new user through the website
2. Access MongoDB:
```bash
mongosh
use cricketdb
db.users.updateOne(
  { email: "your-email@example.com" },
  { $set: { isAdmin: true } }
)
```
3. Logout and login again
4. Access Admin Panel from navigation

### Adding Sample Data

Use the sample data provided in `SAMPLE_DATA.md` or add through Admin Panel:

```bash
mongosh
use cricketdb

# Insert teams
db.teams.insertMany([...])

# Insert players
db.players.insertMany([...])

# Insert matches
db.matches.insertMany([...])
```

## ğŸ“¡ API Endpoints

### Authentication
```
POST   /api/auth/register    Register new user
POST   /api/auth/login       Login user
```

### Players
```
GET    /api/players          Get all players
POST   /api/players          Add new player (Admin)
PUT    /api/players/:id      Update player (Admin)
DELETE /api/players/:id      Delete player (Admin)
```

### Teams
```
GET    /api/teams            Get all teams
POST   /api/teams            Add new team (Admin)
PUT    /api/teams/:id        Update team (Admin)
DELETE /api/teams/:id        Delete team (Admin)
```

### Matches
```
GET    /api/matches          Get all matches
POST   /api/matches          Add new match (Admin)
PUT    /api/matches/:id      Update match (Admin)
DELETE /api/matches/:id      Delete match (Admin)
```
